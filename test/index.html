Welcome to the FoxSports.com.au front-end JQuery test. In this test, we are looking at your ability to write elegant, efficient and semantically correct code (JS, XHTML and CSS) that is compatible with the latest public versions of Firefox, Google Chrome and Safari, as well as Internet Explorer 8 and 9. Please also be aware that Chrome can't deal with local json files, so part 2 won't work in Chrome (unless you are running it through a local webserver).

  • The json-module
